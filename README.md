### ⚙️ Deep Dive: Browser Architecture & Optimization
Every line of code I write is optimized for the **Critical Rendering Path (CRP)**:
1. **Parsing Optimization:** Semantic HTML for rapid DOM construction + Lightweight CSSOM.
2. **Reflow & Repaint Prevention:** Active avoidance of layout-triggering properties to prevent thrashing.
3. **GPU Compositing:** Leveraging hardware acceleration via `transform` and `opacity` for 60FPS UI.
4. **Zero-JS State:** Utilizing `:has()`, `peer`, and CSS State Lifting to minimize main-thread execution.
