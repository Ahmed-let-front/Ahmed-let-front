### ⚙️ Under The Hood: How I Optimize

1. **DOM & CSSOM Efficiency:** I write shallow, highly semantic HTML. By utilizing utility frameworks (Tailwind) and purging unused styles, I prevent the CSS engine from blocking the main thread.
2. **Reflow & Repaint Prevention:** I actively avoid animating layout-triggering properties (`width`, `margin`). I push animations to the Compositor Thread using `transform` and `opacity` to guarantee a rock-solid **60 FPS**.
3. **Strict W3C Semantics:** Utilizing the "Container Bag" mental model to guarantee 100% Screen Reader accessibility natively.
